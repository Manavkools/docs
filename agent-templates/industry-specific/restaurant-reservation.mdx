## Overview

The Restaurant Reservation Agent handles reservation requests, confirmations, and modifications for dining establishments.

## Use Cases

- Restaurants
- Cafes
- Catering services
- Hotels with dining

## Key Features

- Reservation booking
- Table availability
- Special requests
- Confirmation calls
- Cancellation handling
- Waitlist management
