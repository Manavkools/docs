## Overview

The Loan Application Assistant Agent guides customers through loan applications, collects information, and answers questions.

## Use Cases

- Banks
- NBFCs (Non-Banking Financial Companies)
- Fintech lenders
- Microfinance institutions

## Key Features

- Application guidance
- Information collection
- Document requirements
- Answer questions
- Status updates
- Compliance handling
